Welcome to the G++ Server

Powered by a custom HTTP server written in C++

Multithread server using raw C++ and is fully optimized to handle your requests efficiently.

Instructions

To access the homepage: Simply navigate to http://localhost:8082/ in your browser. The server will automatically serve the index.html file.

To view images: You can access any image served by the server by visiting its URL directly. For example, to view the C++ logo, go to http://localhost:8082/img/cpp-logo.png.

To access the CSS stylesheet: The stylesheet can be viewed by navigating to http://localhost:8082/style.css. This file contains the styles applied to the webpage.

To add more files: Place the files (HTML, images, CSS, JS, etc.) in the project1_httpserver folder. Once added, these files can be accessed directly by navigating to http://localhost:8082/filename.
